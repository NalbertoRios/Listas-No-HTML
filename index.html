Tabalhando com listas

Vamos aprender sobre as listas ordenadas,listas não ordenadas e listas de definição.

Listas ordenadas

Para fazer as listas ordenadas uilizamos a tag <ol>:

  1. Acordar
  2. ligar para João
  3. Tomar café
  4. Escovar os dentes
  5. Ir para o trbalho
  6. Voltar para casa
  7. Jantar
  8. Dormir

    9. Podemos também utilizar o atributo type="" com A,a,I para ordenarem a lista e também start="um número" para iniciar a partir desse número.

    Ex:

    1. Acordar
    2. ligar para João
    3. Tomar café
    4. Escovar os dentes
    5. Ir para o trbalho
    6. Voltar para casa
    7. Jantar
    8. Dormir

      9. Ou

      1. Acordar
      2. ligar para João
